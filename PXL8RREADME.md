1. Load an Image

Drag and drop an image onto the drop zone
or

Click the drop zone and select a file

Your image will appear in two panels:

Original Image

Pixelated Output

2. Pixelate the Entire Image

Use the FULL IMAGE controls:

Pixel Size

Adjusts how blocky the pixelation is

Higher numbers = bigger pixels

Color Depth

Limits the number of colors

Lower numbers = more retro/flat colors

Retro Palettes

Choose a classic console palette:

Game Boy

NES

SNES

GBA
or select NONE for normal pixel reduction.

These palettes give authentic old-school color styles.

3. Pixelate Only a Selected Area

This is the easiest way to censor part of an image or hide a thumbnail.

Click SELECT

Click and drag on the image to draw a rectangle

Adjust the SELECTION pixel size + color depth

Press APPLY

Only that selected area will be pixelated.
You can repeat this as many times as needed.

4. Reset the Image

Click RESET to revert back to the original, loaded image.

5. Save Your Result

Click SAVE to download the pixelated image as a PNG.
